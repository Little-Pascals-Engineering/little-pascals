How to Use the Calculator:

To use the calculator with the present values, simply make sure that the file “TLP_Project2_Calculator” is next to the data folder. In Jupyter Notebook, open the calculator, click “cell” in the nav bar, and then click “run all”.

If you wish to enter more data into the .csv files, do so in the same format they are currently in written in.
