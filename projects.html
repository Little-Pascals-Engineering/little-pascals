OHSU Contingency Plan

Project Overview:

For this project we were tasked with establishing which option of four different local reservoirs was cheapest for getting water to the OHSU campus. We also considerd how much the cost would be if the piping between each reservoir and OHSU were to be restored.

Truck Engine Fluid Pump

Project Overview:

In this project, we were tasked with figuring out which pump system would be best in puhsing oil through a new engine system for Fake Trucks Inc. To figure out which pump was best, we factored in variables like oil used, power used, pipe diameter, and flow rate in each of the major sections.

Reynolds Boiler System Replacement

Project Overview:

For this project we were tasked with replacing the boilers and retrofiting the piping in the Reynolds High School boiler room. This involved replacing a large amount of existing piping on a tight budget so that the school's centeral air could function properly.

Columbia Wastewater Treatment Facility

Project Overview:

At the The Columbia Boulevard Wastewater Treatment Plant we were tasked with doing a comprehensive maintainence overview. We checked all of the piping against its orignal design specifications to see what pipes were still working well and what needed to be replaced.
